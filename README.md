# 计时器使用指南（非开发者）

本工具用于课堂辩论赛计时与流程展示，支持直接双击打开使用，无需安装任何软件。

## 一、打开方式

1. 解压/打开文件夹
2. 双击 `index.html` 即可打开

## 二、切换辩题组（从 CSV 导入）

1. 准备好 `data.csv`
2. 打开页面后，点击“加载CSV”
3. 在下拉框中选择辩题组
4. 点击“应用”，页面将更新正反方队名与辩题

### data.csv 格式

表头必须包含以下列（顺序可不一致）：

- `序号`
- `正方队名`
- `正方辩题`
- `反方辩题`
- `反方队名`

每一行是一组辩题，例如：

```
序号,正方队名,正方辩题,反方辩题,反方队名
1,第一组,观点A,观点B,第四组
2,第二组,观点C,观点D,第五组
```

提示：也支持制表符分隔（从 Excel 另存为 CSV/TSV 均可）。

## 三、调整阶段时间与总标题（config.js）

打开 `config.js`，可改以下内容：

- `title`：页面顶部总标题
- `schedule`：阶段列表与时长

### 自由辩时间

自由辩阶段的 `duration` 表示“单方时长（秒）”，系统会自动乘以 2 作为总时长。

示例：

```
{ group: "自由辩阶段", label: "自由辩论", side: "双方", duration: 360, free: true }
```

表示正方 360 秒 + 反方 360 秒。

## 四、常用按键

- 空格：开始/暂停
- ← →：上一环节 / 下一环节
- R：重置
- ↑ ↓：±10 秒
- F / J：自由辩时切换正方/反方发言

## 五、常见问题

- 看不到数据：请确认 `data.csv` 在电脑中可被选中并成功导入
- 修改配置后没生效：请刷新页面
